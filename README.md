# 📌 Project Overview

> *Need-Fund is a medical crowdfunding platform where people facing genuine medical emergencies can request financial help while donors contribute with confidence through a verified, admin-controlled review process.*

The platform focuses on one major problem:

### **Trust**

Many patients genuinely need help.

Many donors genuinely want to help.

But fraud destroys trust.

Need-Fund bridges that gap.

---

# ❌ The Problem

Millions of families struggle to afford treatment.

```text
🏥 Medical Emergency
          │
          ▼
💸 Treatment Cost Too High
          │
          ▼
📢 Family Requests Help
          │
          ▼
🤔 Donors Fear Fraud
          │
          ▼
❌ Support Never Arrives
```

Without verification, donors often hesitate to contribute.

# ✅ The Solution

Need-Fund creates a verified crowdfunding workflow.

```text
👤 Patient Submits Case

          ▼
```

# 🛡️ Anti-Fraud Verification Engine

Fraud prevention is the heart of Need-Fund.

```text
📄 Case Submitted
        │
        ▼
🛡️ Admin Review
        │
   ┌────┴────┐
   ▼         ▼

✅ Genuine   ❌ Fake

   ▼         ▼

Publish    Reject
```

### Verification Rules

| Security Layer             | Status |
| -------------------------- | ------ |
| Manual Admin Review        | ✅      |
| Medical Proof Check        | ✅      |
| Fake Case Rejection        | ✅      |
