Need-Fund 🏥
Trusted Medical Crowdfunding Platform
A platform where people in genuine medical need can share their situation and receive financial help from donors — with a verified, admin-controlled system to prevent fraud.

💡 The Idea
Hospitals have patients who cannot afford treatment. Wealthy or kind-hearted people want to help but don't trust random requests. Need-Fund bridges this gap — every case is verified by an admin before going live, so donors give with full confidence.

✨ How It Works
Patient submits their case (medical details, photos)
      ↓
Admin reviews and verifies authenticity
      ↓
Verified case goes live on platform
      ↓
Donors browse verified cases
      ↓
Donor contributes directly to the patient ✅

🔒 Anti-Fraud System

Every case manually reviewed by admin before publishing
Fake or unverified cases never go live
Admin can remove suspicious listings anytime
Donor trust maintained through verification badge
